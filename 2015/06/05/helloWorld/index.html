First Introduction to Markdown

Posted on 2015-06-05

Markdown - Grammar

Title

To define a title, put “#” in front. Different numbers of “#” determines how big the title is. The font gets smaller as the number of “#” increases.

For example: 

[markdown] #This is a line of text

this is a line of text

[markdown] ###This is another line of text

This is another line of text


List

To define an unordered list, put “-“ or “*” in front.

For example: 

[markdown] * Tokyo
           * Hongkong
           * Shenzhen
           * Manila
  • Tokyo
  • Hongkong
  • Shenzhen
  • Manila

Notice that “*” has to be followed by a space then the list item

To define an ordered list, put “1.”,”2.”,”3.” in front of the list items.

For example:

[markdown] 1. apple
           2. kiwi
           3. orange
  1. apple
  2. kiwi
  3. orange

Quotes

To introduce a quote, put “>” in front

For example:

[makrdown] >To be or not to be this is a question

To be or not to be this is a question

Picture & Link

To include a picture, use the following format:

[pictureName](Picturelink)
